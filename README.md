# Private Posts Application

**Author:** Zachary Jolliff  
**Project:** Project 3 – Private Posts

## Short Description

Private Posts is a simple Ruby on Rails web application that implements basic authentication and CRUD functionality. Users can sign up, log in, create new posts, view a list of posts, edit or delete their own posts, and log out. The home page displays posts and serves as the main entry point to the app’s features.
